# 111youtubeToText

將指定直播的網址語音轉文字


排程：
　　1、注意，首次使用請先下載「vosk」：https://alphacephei.com/vosk/models/vosk-model-cn-0.1.zip
解壓縮後放到 binary/vosk/model

　　2、App.config 裡 tmpPath 與 mp4Path 建議指派到 ramdisk 作處理，請下載安裝
    https://3wa.tw/uploads/upload/ramdisk_setup.exe
    建議可配置在 R:，分格區請選 ntfs
